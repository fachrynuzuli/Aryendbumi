# Cheat Code Untuk Keep Calm

[Original date: 19 Jul, 2024 12:13]

"Keep calm and carry on."

Kalau kamu seumuranku, trend ini ga bakal terlewatkan olehmu. "Carry on" yang diganti menjadi 1.000+ alternatif lain untuk mendukung apapun konsep di kepala.

- Overthinking → Nulis
- Ga punya inspirasi → Baca
- Lelah → Tidur
- Bingung → Bertanya
- Frustrasi → Bergerak
- Rasa takut → Ambil risiko
- Pikiran berkabut → Jalan kaki
- Burned out → Take a day off
- Ga sabaran → Review progres
- Zero motivation → Ingat "Why" kamu

Keep calm and JUST fucking do it.
